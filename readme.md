xpathtool
=========

Tool to test XPath syntaxe. It's build with [qooxdoo] framework

=======
>Tool to test XPath syntaxe. It's build with [qooxdoo] framework

---

##Goal##

>Validate XPath syntax easily:

>* __Load__ XML file
* __Show__ file in context
* __Detect__ namespace
* __Create__ XPath
* __Save__ XPath in memory list
* __Show__ the result















































[qooxdoo]: http://qooxdoo.org
